XJson规定如下：

  1. 一般情况下属性和子节点都对应于对象属性
  2. 如果节点上标记了j:list=’true’表示当前节点对应于一个列表对象
  3. 如果节点名为 _，则表示节点名被忽略
  4. 如果节点没有属性，且没有子节点，则它的内容作为它的值
  5. 如果节点具有j:key属性，则它替代节点名作为属性名
  6. 列表节点的子节点的tagName对应于type属性。但有个例外，如果节点名为 _，则表示节点名被忽略
